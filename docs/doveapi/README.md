# 🕊 DoveAPI - 安装框架

~~通过composer安装框架：~~
```composer
composer create-project xcenweb/doveapi ./
```

⚠️目前推荐直接安装开发版本的框架⚠️

```composer
composer create-project xcenweb/doveapi:dev-main ./
```

更新框架：

```composer
composer update xcenweb/doveapi
```
